How Icons Were Sourced

Documentation of the icon extraction process for the Windows App Tray. Every icon is real — no generated placeholders.

1

Identify source executables

Windows applications embed their icons directly in the .exe file. We locate each application's executable path:

File Explorer → C:\Windows\explorer.exe
Microsoft Edge → C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe
Google Chrome → C:\Program Files\Google\Chrome\Application\chrome.exe
Comet Browser → C:\Users\[user]\AppData\Local\Perplexity\Comet\Application\comet.exe
2

Extract icons from .exe files

Used win32gui.ExtractIconEx() from the pywin32 library to extract the primary icon from each executable as an HICON handle:

import win32gui, win32ui
large, small = win32gui.ExtractIconEx(exe_path, 0, 1)
hicon = large[0] # First large icon

# Create a compatible DC and bitmap
hdc = win32ui.CreateDCFromHandle(win32gui.GetDC(0))
hbmp = win32ui.CreateBitmap()
hbmp.CreateCompatibleBitmap(hdc, 64, 64)

# Draw the icon onto the bitmap
win32gui.DrawIconEx(hdc2.GetSafeHdc(), 0, 0, hicon, 64, 64, 0, 0, 0x0003)

# Convert to PIL Image via raw BGRA buffer
img = Image.frombuffer('RGBA', (64, 64), bmpstr, 'raw', 'BGRA', 0, 1)
3

Handle web-only apps (Perplexity)

Perplexity doesn't have a native Windows .exe — it's a web app. We fetched its favicon from the official domain:

Source: https://www.perplexity.ai/favicon.ico
Downloaded: 14.7KB .ico file
Contains sizes: 16x16, 32x32, 48x48
Resized to 64x64 using PIL LANCZOS resampling
Saved as perplexity.png and perplexity.ico
4

Save in dual format

Each icon saved as both .ico (Windows native) and .png (universal fallback):

File Explorer native
Microsoft Edge native
Google Chrome native
Perplexity favicon
Comet Browser native
5

Load into system tray

The app reads the .ico files at startup and creates a pystray Icon for each. Real Windows icons, real tray presence.

from PIL import Image
import pystray

icon_img = Image.open("icons/chrome.ico")
tray = pystray.Icon("chrome", icon_img, "Google Chrome", menu)
tray.run()
